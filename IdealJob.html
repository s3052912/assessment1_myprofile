Ideal Job

My ideal job would be a Cloud Architect. The Job advertisement attached below is looking for a Cloud solutions Architect at Accenture.

This position is looking for an employee with a deep level of understanding of one of the major global cloud platforms – AWS, Azure and Google. Working as a consultant, the employee in this position would work with their clients to design and implement a cloud based solution to meet the clients needs. The appealing part of this job is the way you can transform the clients infrastructure to keep up with ever evolving technologies. It’s also appealing that with more and more SaaS, PaaS and IaaS services available nowadays, I see great future job security in this field.

The skills required for this job are cloud networking skills, programming, database skills, cybersecurity. Experience in cloud infrastructure tools such as Automation tools to provision, testing, deployment requirements, and Integration tools to integrate various solutions. A good working knowledge in Agile Project Delivery methodologies.

Along with learning Programming in the Bachelor of IT course, I also plan to take the Cloud Computing and Cloud Security electives as a part of this course.
